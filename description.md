# Project Technologies

Dự án này sử dụng các công nghệ và thư viện chính sau dựa trên `package.json`:

## 1. Core Framework & Library
- **Next.js (v14.2.3)**: React framework được sử dụng để xây dựng ứng dụng với các tính năng như Server-Side Rendering (SSR), App Router.
- **React (v18)**: Thư viện cốt lõi để xây dựng giao diện người dùng (UI).
- **TypeScript (v5)**: Ngôn ngữ lập trình chính, cung cấp static typing giúp tăng tính an toàn và dễ bảo trì cho mã nguồn.

## 2. Styling & UI
- **Tailwind CSS (v3.4.1)**: Utility-first CSS framework để styling ứng dụng một cách nhanh chóng và linh hoạt.
- **clsx** & **tailwind-merge**: Các tiện ích hỗ trợ quản lý, kết hợp các class name của Tailwind CSS một cách tối ưu, tránh xung đột.
- **Lucide React**: Thư viện icon mã nguồn mở với thiết kế hiện đại.

## 3. State Management
- **Zustand**: Thư viện quản lý state (trạng thái) toàn cục nhỏ gọn, hiệu suất cao và dễ sử dụng cho React.

## 4. Data Visualization
- **Recharts**: Thư viện biểu đồ (chart) mạnh mẽ dành cho React, giúp trực quan hóa dữ liệu dễ dàng (phù hợp cho các dashboard).

## 5. Tooling & Linter
- **ESLint**: Công cụ phân tích mã nguồn để đảm bảo code chuẩn chỉ và tìm kiếm các lỗi tiềm ẩn.
- **PostCSS**: Công cụ xử lý CSS, thường đi kèm với Tailwind CSS.
